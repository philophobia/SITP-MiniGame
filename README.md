# SITP-MiniGame
安卓项目的一部分，当做练习，非常简单且基础。

## 内容
- “占花名”：行酒令游戏
- “猜灯谜”：普通的猜灯谜游戏

## 知识点
- 随机数
- API 21以上含有的过渡动画Explode、Slide、Fade
- 使用xml建立动画

## 遗憾/改进
找到了一个开源项目[模糊了背景的对话框](https://github.com/tvbarthel/BlurDialogFragment)，感觉可以用，但是对Fragment
不熟悉导致在这个MiniGame中无法达到理想的效果，只能退而求其次用普通的对话框了。
